# AI_Boat_Pilot
It is a boat controlled by a machine learning agent that is tasked to reach the goal as fast as possible.

![alt text](https://i.postimg.cc/qBGczH8F/Screenshot-2026-06-08-200445.png)

[Try it](https://play.unity.com/en/games/f23f8b46-5557-4a9b-ac10-e57c9501651e/aiboatpilot11)

__What can it do?__

It scans in front of it with shapecasts and collects data

It can row with the right oar to boost its forward velocity while also stearing to the left

It can row with the left oar to boost its forward velocity while also stearing to the right

It can break with the right oar to slow its forward velocity while also stearing to the right

It can break with the left oar to slow its forward velocity while also stearing to the left

__How to run this localy__

To set up this project and start making ML-Agents I followed a tutorials series by Ludic Worlds, there you can find all of the requierements, example projects and how to make your own ML-Agents if you want to try something similar.
